# Features Overview

## Core Features

### 1. Site Blocking
- **Pre-configured Sites**: Facebook, Instagram, Twitter, LinkedIn, TikTok, YouTube, Reddit, Snapchat, Pinterest, WhatsApp Web
- **Custom Sites**: Add any website domain to block
- **Enable/Disable**: Toggle individual sites on/off
- **Smart Blocking**: Automatically redirects when you try to access blocked sites

### 2. Time Management
- **Flexible Durations**: 10 minutes, 30 minutes, 1 hour, 2 hours, 4 hours, 8 hours, 24 hours
- **Auto-Expiry**: Blocking automatically stops after the set duration
- **Extend Block**: Extend blocking time from the block page
- **Real-time Status**: See remaining time in the popup

### 3. Motivational Messages
- **Default Messages**: 8 pre-loaded motivational messages
- **Custom Messages**: Add your own personalized messages
- **Random Display**: Different message each time you're blocked
- **Character Limit**: Messages limited to 200 characters for readability

### 4. Usage Statistics
- **Daily View**: See your progress for the last 30 days
- **Weekly View**: Aggregate data for the last 12 weeks
- **Monthly View**: Monthly summaries for the last 12 months
- **Interactive Charts**: Bar charts showing time saved and blocks
- **Key Metrics**:
  - Total time saved
  - Total blocks prevented
  - Average daily time saved
  - Current streak of productive days

### 5. Data Management
- **Export Data**: Download your usage statistics as JSON
- **Clear Data**: Reset all statistics (with confirmation)
- **Local Storage**: All data stored locally in your browser
- **Privacy**: No data sent to external servers
