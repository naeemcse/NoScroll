# Features Overview

## Core Features

### 1. Site Blocking

**Default Blocked Sites:**
- Facebook
- Instagram
- Twitter/X
- LinkedIn
- TikTok
- YouTube
- Reddit
- Snapchat
- Pinterest
- WhatsApp Web

**Custom Sites:**
- Add any website domain
- Block custom sites
- Manage in settings

**Blocking Method:**
- Blocks newsfeed/content areas only
- Keeps messaging and navigation accessible
- Shows motivational overlay
- Site loads normally, content is hidden

### 2. Access Control

**Temporary Access:**
- Grant access for 5, 10, 30, 60 minutes
- Or grant forever access
- Per-site control
- Revoke anytime

**Access Management:**
- See which sites have access
- View remaining time
- Extend or revoke access
- Independent per site

### 3. Motivational Messages

**Default Messages:**
- 8 pre-loaded messages
- Randomly displayed
- Customizable

**Custom Messages:**
- Add your own messages
- Remove default messages
- Unlimited messages
- Personal motivation

### 4. Usage Statistics

**Time Tracking:**
- Time spent on allowed sites
- Per-domain breakdown
- Daily/weekly/monthly views

**Visit Tracking:**
- Number of times sites opened
- Entry attempts logged
- Access grants tracked

**Visualizations:**
- Interactive bar charts
- Website usage graphs
- Time-based trends
- Filterable data

### 5. Website Usage Graph

**Features:**
- Shows which websites you use most
- Time spent per website
- Horizontal bar chart
- Top 15 websites displayed

**Filters:**
- Today
- Last 7 Days
- Last 30 Days

**Display:**
- Websites on Y-axis
- Time on X-axis
- Tooltips with exact time
- Sorted by usage

## User Interface

### Popup Interface

- **Status Indicator**: Active/Inactive status
- **Today's Usage**: Quick stats
- **Site Access Control**: Manage access
- **Quick Actions**: Stats and settings

### Settings Page

- **Blocked Sites**: Enable/disable sites
- **Custom Sites**: Add custom domains
- **Motivational Messages**: Manage messages
- **Reset Options**: Restore defaults

### Statistics Page

- **Overview Cards**: Key metrics
- **Time Charts**: Daily/weekly/monthly
- **Website Graph**: Domain usage
- **Detailed Stats**: Day-by-day breakdown

## Data Management

### Export Data

- Download all statistics as JSON
- Contains usage data
- Can be imported (future)
- Human-readable format

### Clear Data

- Remove all statistics
- Keep settings intact
- Confirmation required
- Cannot be undone

## Privacy Features

### Local Storage

- All data in browser
- No external servers
- No cookies used
- Extension-only access

### No Tracking

- No analytics
- No data collection
- No external requests
- Complete privacy

## Advanced Features

### Always-On Blocking

- Blocking enabled by default
- No need to start/stop
- Automatic protection
- Persistent across restarts

### Smart Blocking

- Site-specific selectors
- Dynamic content handling
- SPA navigation support
- Mutation observer

### Time Tracking

- Accurate time calculation
- Per-domain tracking
- Session management
- Automatic aggregation
